Include each of your Rezgo tours as custom posts for use in XML sitemaps.

== Description ==

Create posts for each of your Rezgo tours using the custom post type *tours*. The tours will then be available for inclusion in XML sitemaps. This is meant to be used in conjunction with the [Rezgo Online Booking](http://wordpress.org/extend/plugins/rezgo-online-booking/) plugin.

== Installation ==

1. Upload the *rezgo-tour-posts* folder to the */wp-content/plugins/* directory
2. Activate the Rezgo Tour Posts plugin through the 'Plugins' menu in WordPress
3. Run the plugin to create posts from your Rezgo tours

== Frequently Asked Questions ==

= Has this been tested with XML sitemap plugins? =

Yes, this has been tested with the [WordPress SEO](http://wordpress.org/extend/plugins/wordpress-seo/) and [Google XML Sitemaps](http://wordpress.org/extend/plugins/google-sitemap-generator/) plugins
